○ソフト紹介
　本サンプルソースは、技術評論社刊「JavaScript本格入門　〜モダンスタイルに
　よる基礎からAjax・jQueryまで」で紹介されているソースコードを電子ベースで
　提供するものです。
　使用するにあたって必要な設定、及び、必要なコンポーネント類等は一切含んで
　いないので注意してください。
　また、環境設定方法については、書籍本文を参照願います。

○サンプルの検証ブラウザ
　本サンプルは、以下のブラウザで動作検証を行っております。

　　・Firefox 3.6

　　・Opera 10.61
　　・Safari 5.0

○参考図書、参考サイト
　より深くJavaScriptを学習したい、あるいは、その周辺技術に関する理解を補足
　したいという方は、以下のような書籍、サイトを参照することをお勧めします。

　1.独習PHP 第2版（翔泳社）
　　　http://www.wings.msn.to/index.php/-/A-03/978-4-7981-2151-2/
　　　Ajax技術を利用するには、サーバサイド技術の理解は欠かせません。
　　　サーバサイド技術にもさまざまなものがありますが、初学者の方が
　　　まず取り掛かるならば、PHPをお勧めします。

　2.「これならわかる Web標準サイトの作り方 入門の入門」（翔泳社）
　　　http://www.wings.msn.to/index.php/-/A-03/978-4-7981-1329-6/
　　　Web業界では、もはやスタンダードとなりつつある「Web標準」。
　　　この「Web標準」に対応するサイト制作の考え方・作り方・デザインについて、
　　　やさしく読みやすく解説します。
　　　XHTML／CSSやアクセシビリティ＆SEOなどに関する技術解説のみならず、
　　　ナビゲーション／視覚表現のコツやテクニックなども紹介します。

　3.jQuery逆引きリファレンス
　　　http://www.atmarkit.co.jp/fdotnet/jqueryref/index/
　　　第9章やAppendixでも紹介したjQueryの逆引きリファレンスです。
　　　さまざまな局面に対応できるよう、目的別に主要なメソッドがまとまって
　　　いますので、jQueryによる開発を行う場合に活用できます。
